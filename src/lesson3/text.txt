

1. Есть класс банковский счёт (у счёта есть ID, Name владельца, amount денежная сумма, boolean поле isActive (true - активен, false - нет)

2. Написать отдельный класс с методами по работе со счетами

а) Метод первый для передачи денег с одного банксовкого счёта на другой. Принимает на вход
два банковских счёта (с которого и на который идет перевод) и сумму перевода.

- Совершает перевод денег с одного счёта на другой.
- Перед переводом проверяет, что денег для перевода достаточно (если нет то кидает собственное непроверояемое
Exception), а также, что оба счёта активны (поле isActive true) -> если что тут тоже кидает собственное непроверяемое Exception

б) Метод, который принимает список( ArrayList счетов) и возвращает из него список только тех счетов, которые не активны

в) Метод, который принимает список( ArrayList счетов) и ID пользователя для поиска. И возрващает из метода конкретный
счёт по тому ID, который мы подали
